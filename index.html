To add a new scenario for visualization:

  • Add the scenario name to the scenarios.csv definition file in the data folder
  • Create a new folder in the data folder with the scenario name
  • Copy in the required visualization data tables, each with the following required fields:
    • BarChartAndMapData.csv: ZONE, SPATIALAGG, BAR, QUANTITY
    • TimeUseData.csv (not yet supported)
    • TourTracingData.csv (not yet supported)
    • TreeMapData.csv (not yet supported)
    • RadarChartsData.csv (not yet supported)
    • BarChartsData.csv (not yet supported)
    • RailStationsData.csv (future work)
